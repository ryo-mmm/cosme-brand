# サイト制作設計書 (DESIGN.md) - コスメ定期便プロジェクト

## 1. プロジェクト概要
本プロジェクトは、30代〜40代の働く女性をターゲットとした、上質なオーガニックコスメの定期便（サブスクリプション）サイトの開発を目的とする。

## 2. ターゲット・ペルソナ
- **ターゲット:** 都市部で働く30代後半〜40代の女性。
- **ライフスタイル:** 忙しい日々の中で「自分へのご褒美」としてのスキンケア時間を大切にしたい。
- **ニーズ:** - 自分の肌に合ったものをプロ（システム）に選んでほしい。
    - 買い忘れを防ぎたいが、在庫が溜まるのは避けたい（調整の自由度）。
    - 信頼できるオーガニック成分、かつ洗練されたデザイン。

## 3. デザイン・コンセプト
- **キーワード:** 「余白」「静寂」「信頼」「手触り感」
- **ビジュアル:** - **カラーパレット:** - メイン：#F5F5F0 (サンドベージュ) - 安心感と上品さ
        - アクセント：#4A5859 (スレートグレイ) - 知的さと信頼
        - ベース：#FFFFFF (ホワイト) - 清潔感
    - **タイポグラフィ:** - 見出し：セリフ体（明朝系）で高級感を演出。
        - 本文：サンセリフ体（ゴシック系）でスマホでの可読性を確保。
    - **UI:** 余白を広く取り、商品写真が美しく映えるミニマルな構成。

## 4. サイト構成 (サイトマップ)
1. **Top:** ブランドストーリーと診断への誘導。
2. **Skin Diagnosis (Vue.js):** 5〜10問の質問で最適なセットを提案。
3. **Product List/Detail:** 定期便の仕組みと単品購入の価格差を明示。
4. **Checkout:** 会員登録・決済（Stripe連携）。
5. **My Page:** - 次回配送日の変更・スキップボタン（1クリックUI）。
    - 過去の診断履歴とおすすめのケア。

## 5. 技術要件 (システム構成)
### フロントエンド (Vue.js)
- **UI Framework:** Tailwind CSS（レスポンシブ対応の高速化）。
- **State Management:** Pinia（診断データやカート情報の管理）。
- **UX:** 診断画面でのシームレスな画面遷移（SPA）。

### バックエンド (Laravel)
- **Version:** Laravel 11.x
- **Payment:** Laravel Cashier (Stripe) を使用し、サブスクリプションロジックを実装。
- **Security:** 2段階認証、SSL、CSRF対策の徹底。

## 6. 主要機能のロジック
- **定期便スキップ機能:** - 配送予定日の3日前までマイページから「1ヶ月スキップ」を可能にする。
    - データベースの `subscriptions` テーブルの次回課金日を更新する。
- **肌質診断ロジック:** - 回答の合計スコアに基づき「乾燥」「オイリー」「混合」「敏感」の4タイプに分類。
    - タイプごとにLaravelから異なる商品IDをフロントエンドに返す。

## 7. PDCAサイクルによる品質管理
- **Plan:** ターゲット層へのアンケートに基づいたUI設計。
- **Do:** クリーンコードを意識した実装（PSR-12準拠）。
- **Check:** 表示速度計測（Lighthouse）、モバイルユーザビリティテスト。
- **Act:** 離脱率の高いページ（特に診断途中）のUI/UX改善。
